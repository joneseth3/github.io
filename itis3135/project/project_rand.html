Project Overview

  1. The applications purpose is to measure the person who wants a bike fit and then using an algorithm it will suggest the best bike fit for them on the type of bike they have.
  2. The intened users of the website will be cyclists looking for a bike fit.
  3. The content of the website will include a form which the user can fill out and a diagram of the type of bike the user selected. Once the user has filled out the form the program will return a p tag with the sugestions of their bike fit.
